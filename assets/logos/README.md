Add or replace logo files here. The site uses these filenames for the **Experience** section.

| File | Company | Notes |

| `ledger.svg` | Ledger | Replace with official logo. |

## Format

- **Format:** SVG preferred (scalable, small size). PNG also works (e.g. `ledger.png`).
- **Size:** Any; they are displayed at 28×28px in the UI. SVG will scale cleanly.
- **Naming:** Keep the exact filenames above so `index.html` finds them.

If a file is missing or fails to load, the card falls back to initials (LoL, L, Ctrl, S, J).
